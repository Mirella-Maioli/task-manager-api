# Task Manager API

A RESTful API for managing tasks built with Python.

This project was created to practice backend development and API design concepts such as CRUD operations, request handling and structured data management.

## Features

- Create tasks
- View task list
- Update tasks
- Delete tasks

## Technologies

- Python
- FastAPI / Flask
- REST API
- JSON

## Future Improvements

- Authentication
- Database integration
- User management

## How to Run

1. Install dependencies

pip install fastapi uvicorn

2. Run the server

uvicorn main:app --reload

3. Open in browser

http://127.0.0.1:8000/docs
